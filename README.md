### Run Locally

To run **Music Player** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/codewithsadee/music-player.git
```

Windows:

```bash
git clone https://github.com/codewithsadee/music-player.git
```
